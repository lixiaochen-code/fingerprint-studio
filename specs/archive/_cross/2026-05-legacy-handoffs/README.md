# Legacy Handoffs Archive

This archive preserves the historical handoff documents that used to live under `docs/handoffs/`.

Use them for decision archaeology only:

- They explain why routing, script runtime, stealth, profile deletion, and testing choices were made in 2026-05.
- They may mention obsolete paths such as `docs/specs/*`, `AGENT.md`, or old component filenames.
- They may say something is "pending" even when later code or `specs/archive/` completed it.

Current truth should be checked in `specs/baseline/`, later `specs/archive/` entries, and the current codebase.

The handoff index is preserved at [`handoffs/README.md`](handoffs/README.md).
